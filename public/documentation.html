myFlix API Documentation

Welcome to the myFlix API documentation. This API provides access to information about movies.

Base URL

The base URL for all API requests is:

http://localhost:8080

URL Endpoints and Request Methods

Request URL Response
GET all movies /movies List of all movies
GET a single movie by title /movies/title/{title} Details of the specified movie (description, genre, director, image URL, featured)
GET a genre by name/title /genres/{name} Details of the specified genre (description)
GET a director by name /directors/{name} Details of the specified director (bio, birth year, death year)
POST a new user registration /users Created user details
PUT (update) user info /users/{username} Updated user details
POST add a movie to user's favorites /users/{username}/favorites/{movieId} Confirmation text that a movie has been added
DELETE a movie from user's favorites /users/{username}/favorites/{movieId} Confirmation text that a movie has been removed
DELETE user deregistration /users/{username} Confirmation text that a user has been removed
