myFlix API Documentation

Welcome to the myFlix API documentation. This API provides access to information about movies.

Base URL

The base URL for all API requests is:

http://localhost:8080

URL Endpoints and Request Methods

Description URL Method Query Parameters Request Body Response
Return a list of all movies /movies GET None None 
[
  { "title": "The Dark Knight", "director": "Christopher Nolan", "year": 2008 },
  ...
]
Return data about a single movie by title /movies/title/{title} GET {title} None 
{
  "title": "Inception",
  "description": "A thief who steals corporate secrets...",
  "genre": "Science Fiction",
  "director": "Christopher Nolan",
  "imageURL": "http://example.com/image.jpg",
  "featured": true
}
Return data about a genre by name /genres/{name} GET {name} None 
{
  "name": "Thriller",
  "description": "A genre characterized by suspense..."
}
Return data about a director by name /directors/{name} GET {name} None 
{
  "name": "Christopher Nolan",
  "bio": "A British-American film director...",
  "birthYear": 1970,
  "deathYear": null
}
Register a new user /users POST None 
{
  "username": "johndoe",
  "password": "password123",
  "email": "johndoe@example.com",
  "birthday": "1985-05-15"
}
 
{
  "username": "johndoe",
  "email": "johndoe@example.com",
  "birthday": "1985-05-15",
  "favoriteMovies": []
}
Update user info /users/{username} PUT {username} 
{
  "username": "newusername",
  "email": "newemail@example.com"
}
 
{
  "username": "newusername",
  "email": "newemail@example.com",
  "birthday": "1985-05-15",
  "favoriteMovies": []
}
Add a movie to user's favorites /users/{username}/favorites/{movieId} POST {username}, {movieId} None 
{
  "message": "Movie has been added to favorites"
}
Remove a movie from user's favorites /users/{username}/favorites/{movieId} DELETE {username}, {movieId} None 
{
  "message": "Movie has been removed from favorites"
}
Deregister a user /users/{username} DELETE {username} None 
{
  "message": "User has been removed"
}
