myFlix API Documentation

Welcome to the myFlix API documentation. This API provides access to information about movies.

Base URL

The base URL for all API requests is:

http://localhost:8080

Endpoints

GET /

Returns a welcome message.

GET /

Response:

200 OK
Content-Type: text/plain

Welcome to my movie club!

GET /documentation

Returns the API documentation (this page).

GET /documentation

Response:

200 OK
Content-Type: text/html

[HTML content of this documentation page]

GET /movies

Returns a JSON object containing data about the top 10 movies.

GET /movies

Response:

200 OK
Content-Type: application/json
